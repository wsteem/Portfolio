![My Portfolio Demo](![Uploading Apr-05-2026 16-16-37.gif…]())


A dynamic personal portfolio website built with ASP.NET Core that showcases my journey as a Computing Science student at the University of Alberta. This site serves as a central hub for my software projects, automatically fetching and displaying repository data directly from my GitHub profile.

🚀 Features GitHub API Integration: Automatically populates project cards with repository names, descriptions, and primary languages.

Responsive Design: Fully optimized for all screen sizes, from mobile to desktop.

Dark Mode Toggle: User-selectable theme for comfortable viewing.

Clean UI: Minimalist aesthetic focusing on project visibility and ease of navigation.

🛠️ Tech Stack Frontend: HTML5, CSS3, JavaScript

Backend: .NET / C#

Deployment: GitHub Pages / Azure (update this based on where you host it)

📂 Featured Projects The portfolio highlights several key repositories, including:

Gomoku-game: A Python-based implementation of the classic strategy game with a smart AI opponent.

Morse-Code-File-Reader: A utility that parses Morse code text files and converts them to English.

Physics-engine: A CSS-focused project exploring motion and rendering.

LeetCode-Solutions: A repository dedicated to my algorithmic problem-solving progress.
